# Website Builder Templates

## Template Types

1. **Service Business** - For trades/services
   - Featured services
   - Online booking
   - Testimonials

2. **Professional Services** - For consultants
   - Bio/credentials
   - Appointment scheduling
   - Contact form

3. **Retail** - For product sales
   - Product gallery
   - Online ordering
   - Location map

## Customization

- Color schemes
- Font selection
- Layout options
- SEO settings

## Implementation

- Templates stored in `/templates` directory
- Static site generation
- Hosted on Cloudflare Pages

## Integration

- Auto-populates from business info
- Syncs with booking system
- Emma AI can suggest content
